OurWeb.exe

About This Project

Team

A collaborative effort with Ha Minh Hoang, Nguyen Bao Han.

My role involved sketching initial concepts and designing the website layout using Figma.

Assignment

Work on a single-page interactive homepage for a festival or an event. The design must includes at least 8 sections, with interactive elements and a navigation system. The submission is required to include both desktop and mobile (iOS) version of the same website structure.

Concept

Internet privacy is a growing concern among the younger generations as hacks and scams become more sophisticated. Our group project aims to educate on the emerging terrain of digital privacy through an interactive and dynamic experience. Visually, the event represents a unique combination of early internet aesthetics blended with modern philosophies of web design to create an active, retro-but functional space that embodies the ongoing battle between old ideals and upcoming challenges in internet privacy

Tools Used

  • Adobe Illustrator
  • Adobe Photoshop
  • Adobe InDesign
  • Figma
